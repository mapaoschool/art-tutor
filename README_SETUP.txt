# Ma Pao Tutor — PWA Desktop/Home Screen Setup
## Files to upload to your GitHub repo (mapaoschool/art-tutor)

### Required files:
1. **index.html** — Replace your existing file with this updated version
2. **manifest.json** — Place in the ROOT of the repo (same level as index.html)
3. **icons/** folder — Upload the entire icons/ folder with all 7 PNG files inside

### Your repo structure should look like:
```
art-tutor/
├── index.html          ← updated (has manifest link)
├── manifest.json       ← NEW
└── icons/
    ├── icon-72x72.png
    ├── icon-96x96.png
    ├── icon-128x128.png
    ├── icon-144x144.png
    ├── icon-152x152.png
    ├── icon-192x192.png
    └── icon-512x512.png
```

## HOW TO INSTALL ON EACH DEVICE

### 📱 iPhone (Safari required)
1. Open Safari → go to your app URL
2. Tap the Share button (box with arrow at bottom)
3. Scroll down → tap "Add to Home Screen"
4. Name: "Ma Pao Tutor" → tap Add
→ Gold icon appears on home screen. Tap to open full-screen.

### 📱 Android (Chrome)
1. Open Chrome → go to app URL
2. Tap ⋮ menu → "Add to Home screen" (or look for install banner)
3. Tap Add
→ Icon appears on home screen. Launches full-screen.

### 🖥️ Windows (Chrome or Edge)
1. Open Chrome or Edge → go to app URL
2. Look for ⊕ install icon in address bar → click it
   OR: ⋮ menu → "Install Ma Pao's Homeschool Tutor"
3. Click Install
→ Launches as standalone app with taskbar icon.

### 🖥️ Ubuntu Linux (Chrome)
Same as Windows — install from Chrome address bar.

OR manually install the shortcut:
1. Download MaPaoTutor.desktop to your home folder
2. Edit the Icon= line to point to where you saved icon-192x192.png
3. Right-click file → Properties → mark as executable
4. Drag to desktop or panel

### 🖥️ Mac OS X Tiger
1. Download MaPaoTutor_MacTiger.command
2. Double-click to run (opens in Safari)
3. To add to Dock: drag the .command file to the Dock
